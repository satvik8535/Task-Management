# Project Statement: Personal Task Manager

## Problem Statement
In today's fast-paced world, individuals struggle to keep track of their daily tasks, priorities, and deadlines. This leads to missed deadlines, poor time management, and increased stress.

## Scope
The Personal Task Manager is a console-based application that allows users to:
- Create and organize tasks
- Set priorities and due dates
- Track completion status
- View task statistics

## Target Users
- Students managing assignments and projects
- Professionals tracking work tasks
- Anyone needing personal task organization

## High-level Features
- Task creation with validation
- Priority-based task organization
- Due date management
- Completion tracking
- Data persistence
- Statistical overview
