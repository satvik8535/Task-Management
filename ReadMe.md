# Personal Task Manager

A Java-based console application for managing personal tasks efficiently.

## Overview
This application helps users organize, track, and manage their daily tasks with features like priority management, due dates, and completion tracking.

## Features
- Add new tasks with title, description, priority, and due date
- View all tasks or filter by priority/status
- Mark tasks as completed
- Delete tasks
- Persistent data storage
- Task statistics

## Technologies Used
- Java 8+
- Object-Oriented Programming
- File I/O for data persistence
- JUnit for testing
- Git for version control

## Installation & Setup
1. Ensure you have Java 8 or higher installed
2. Clone the repository: `git clone <repository-url>`
3. Navigate to the project directory
4. Compile: `javac -d out src/main/java/**/*.java`
5. Run: `java -cp out Main`

## Testing
Run tests with: `java -cp out org.junit.runner.JUnitCore service.TaskServiceTest`
